# 《司马光》原文材料与教学解析

本文件用于 `WENYAN-GENERALIZE-001-RUN` 的 Phase 2 原文材料记录。当前文本为烟测默认文本，正式教材版本仍待确认。

## 标题

《司马光》

## 本轮默认原文

```text
群儿戏于庭，一儿登瓮，足跌没水中。众皆弃去，光持石击瓮破之，水迸，儿得活。
```

## 白话释义

一群小孩在庭院里玩耍，一个孩子爬上大瓮，不小心掉进瓮里的水中。其他孩子都惊慌离开了，司马光拿起石头砸破瓮，水流出来，掉进瓮里的孩子得救了。

## 关键字词解释

| 原文词句 | 白话解释 | 教学提示 |
|---|---|---|
| 群儿 | 一群孩子 | 建立人物群体，不写成成人事件。 |
| 戏于庭 | 在庭院里玩耍 | “于”表示地点。 |
| 一儿登瓮 | 一个孩子爬上大瓮 | “登”是爬上、登上。 |
| 足跌没水中 | 脚滑跌落，没入水中 | 危机来自原文，不夸张成恐怖场面。 |
| 众皆弃去 | 众人都离开了 | 体现惊慌和对比，不写成恶意逃跑。 |
| 光持石 | 司马光拿起石头 | 体现判断后的行动。 |
| 击瓮破之 | 击破大瓮 | “之”指大瓮。 |
| 水迸 | 水涌出 | 表现解困结果。 |
| 儿得活 | 孩子得救 | 完成事件结果。 |

## 事件顺序拆解

1. 群儿戏于庭：一群孩子在庭院游戏。
2. 一儿登瓮：一个孩子爬上大瓮。
3. 足跌没水中：孩子失足跌入瓮中水里。
4. 众皆弃去：其他孩子惊慌离开。
5. 光持石：司马光没有跟着离开，而是拿起石头。
6. 击瓮破之：司马光击破大瓮。
7. 水迸，儿得活：水流出，孩子得救。

## 教学点

- 按事件顺序复述原文。
- 理解“于”“登”“没”“皆”“弃”“持”“击”“之”“迸”“得活”等词。
- 理解危机中“众皆弃去”和“光持石击瓮破之”的对比。
- 理解司马光的冷静、判断和行动力。
- 理解解决问题时可以改变方法：救人不一定先拉人，也可以先让水流出。
