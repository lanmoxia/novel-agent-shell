# 《司马光》事件危机型结构与 Scene 映射

本文件用于 Phase 3 情境复演结构与 Scene 原文映射。当前项目已推进到 Phase 4 烟测草案，但尚未进入 Phase 5 教学与生产审查。

## 结构目标

- 总时长：约 60 秒烟测结构，最终时长待确认。
- Scene 数量：5。
- 主要场景：古风庭院。
- 结构类型：事件危机型 / 行动解困型。
- 教学闭环：孩童游戏 -> 坠瓮危机 -> 众人惊慌 -> 司马光击瓮 -> 水出人活 -> 回扣原文。

## 避免套用争辩型结构

本项目不得机械套用《两小儿辩日》的争辩型结构。

- 不设置两个观点阵营。
- 不设计观点对错辩论。
- 不用“观点 + 证据”作为主结构。
- 不让角色争论救法长篇辩难。

本项目应使用：

```text
入古境
→ 见孩童游戏
→ 突发坠瓮危机
→ 众人惊慌
→ 司马光判断
→ 击瓮救人
→ 回扣原文
→ 明方法与品质
```

## Scene 顺序说明

- 开场 Scene：庭院游戏，建立“群儿戏于庭”。
- 发展 Scene：一儿登瓮后失足落水，危机由原文触发。
- 转折 Scene：众皆弃去，形成群体惊慌与司马光冷静的对比。
- 行动 Scene：光持石击瓮破之，突出判断与行动。
- 学习闭环 Scene：水迸，儿得活，回扣原文并引导按事件顺序复述。

## 关键对白方向

本文件不写锁稿级完整台词，只记录关键对白方向：

- S01：用题目钩子引出“庭院里发生了什么事？”
- S02：用短句提示“有人跌进瓮里了”，避免惊悚叫喊堆叠。
- S03：突出“众皆弃去”的惊慌反应，不进行道德羞辱。
- S04：司马光用简短判断表达“先让水出来”，不讲现代救援课程。
- S05：用原文和白话回扣“水迸，儿得活”，让学生按顺序复述。

## 字幕要点汇总

本轮不生成正式字幕文件，只记录烟测字幕要点：

- 必须出现的原文：
  - 群儿戏于庭。
  - 一儿登瓮，足跌没水中。
  - 众皆弃去。
  - 光持石击瓮破之。
  - 水迸，儿得活。
- 必须出现的白话提示：
  - 一群孩子在庭院里玩耍。
  - 一个孩子失足跌入瓮中水里。
  - 大家惊慌离开，司马光留下来想办法。
  - 司马光击破大瓮，让水流出来。
  - 孩子得救，按事件顺序复述原文。
- 不应出现的解释：
  - 现代救援技术长讲解。
  - 对其他孩子的道德审判。
  - 司马光超能力或武侠式表现。
