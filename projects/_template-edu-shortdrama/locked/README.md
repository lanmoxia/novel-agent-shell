# 生产稿包锁定区

本目录用于 Phase 7。`FINAL-SCRIPT.md` 只代表最终剧本，不代表完整生产稿包。

完整生产稿包还应包含：

- 原文映射表
- 字幕方案
- 分镜方案
- 视频风险说明
- 审查结论
- 修订记录
- Skill 锁定信息
- manifest

## 锁定规则

- 未经用户确认不得锁稿。
- 审查未通过不得锁稿。
- 修订未完成不得锁稿。
- 锁定后不得自动修改生产稿包。
