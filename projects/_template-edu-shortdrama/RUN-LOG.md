# 运行日志

本文件记录项目级执行日志、用户确认和关键状态变化。它不是长期规则库，也不替代 `PROJECT-STATE.json` 或 `PROJECT-MEMORY.md`。

## 记录规则

- 每次 Phase 推进、用户确认、审查、修订、锁稿和后置输出都应追加记录。
- 不记录 `wenyan-skill` 的全文规则，只记录本项目发生的事实和决策。
- 不得用日志记录伪造完成状态；状态必须与实际产物一致。
