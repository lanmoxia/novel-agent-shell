### E01 - 尚仪局旧臣女入宫，太和殿遗诏印存疑

- **章节定位**：起——权局与困境
- **核心事件**：
  - 沈时安以"罪臣之女外戚远亲"身份被召入宫，任尚仪局女史（掌礼仪文书）
  - 入宫当日恰逢太和殿议登基仪轨——三方当场对峙：萧太后出示遗诏（立幼帝），徐首辅指出押印与旧档印文不合，卫将军以"无真诏不调兵"拒调禁军
  - 沈时安在旁记录仪轨，旁听到遗诏押印之争，隐约意识到此事与父亲旧案有关
  - 入夜，方尚仪私下让她整理沈怀述案旧档——"这些旧档搁了两年，该有人翻了"
- **出场人物**：沈时安（主角登场）、方尚仪（引路人）、萧太后（掌遗诏者）、徐首辅（质疑者）、卫将军（消息中——沈时安入宫前夜收到其密信："宫中有人要见你"）、孙掌印（遗诏保管者，仅出场一句——"此印乃先帝亲押"）
- **权力冲突**：
  - 太后 vs 首辅：遗诏押印真伪之争（太后称"先帝亲押"，首辅称"印文不合"）
  - 文官 vs 武将：首辅要调兵维持登基秩序，卫将军以"无真诏不调兵"为由拒绝——兵权在握者的中立是最大的威胁
- **信息不对称**：
  - 沈时安知道：父亲留下半枚勘合符 + 一封未拆密信
  - 太后不知道：沈时安手中有勘合符
  - 首辅知道：遗诏押印与旧档不符
  - 所有人不知道：真诏藏于何处
- **证物推进**：
  - 遗诏首次出现（押印存疑）
  - 勘合符半枚（沈时安入宫前将其缝入衣襟夹层）
  - 沈怀述案旧档（方尚仪交付）
- **谋局/反制**：
  - 太后用遗诏（伪）推进登基程序——以时间压力逼迫各方服从
  - 首辅以礼制程序（押印不合祖制）反制——拖延时间争取查证
  - 卫将军以"无真诏不调兵"反制双方——保持中立以守兵权
- **情绪节点**：
  - 沈时安入宫：克制中的不安（"宫门在她身后合上，铜钉冰凉"）
  - 太和殿对峙：权力压迫感（"殿中烛火被穿堂风带得一晃，无人抬头"）
  - 方尚仪递旧档：暗中的善意（"她放下卷宗就走，没有回头"）
- **章末钩子**：沈时安回到值房，拆开父亲留下的那封密信——信末一行："真诏在旧邸，切莫轻示于人。"她盯着这行字，手指在勘合符的铜边上按出印痕。门外传来脚步声。
- **与 STORY-SPEC 对应**：
  - 主角设定 §5（沈时安身份/处境/性格标签）
  - 外部冲突 §7（三方势力对立 + 时间压力 + 礼制制约）
  - 信息不对称 §8（各方知/不知表）
  - 证物锚点 §9（勘合符 + 旧档）

### E02 - 阅秘档勘合符露隙，设夜宴三方各探锋

- **章节定位**：承——谋局与试探
- **核心事件**：
  - 沈时安借整理旧档之机，查阅司礼监押印秘档——发现遗诏所用印文与国玺不符，却与一种内廷私印的印文字形吻合。她取出勘合符比对：符上缺口处的印文，与遗诏印文出自同一枚私印
  - 萧太后设中秋夜宴，名为赏月，实为试探。席间太后问沈时安"你父亲可有留下什么"，沈时安以"婢子年幼，先父不曾托付"回应——太后眼神微变
  - 徐首辅席间借敬酒之机对沈时安说："尚仪局旧档，阅了便罢，莫要带出。"——他在暗示他知道她在查
  - 卫将军席散后在廊下与沈时安短暂碰面——"你父亲当年不是私匿文书，是看见了不该看见的东西。"
- **出场人物**：沈时安、方尚仪（旁观者——席间一直留意沈时安）、萧太后（试探者）、徐首辅（警告者）、孙掌印（席间沉默——只在太后问话时答"是"）、卫将军（短暂碰面）
- **权力冲突**：
  - 太后设宴试探：想知道沈时安手中是否有沈怀述遗留之物——她担心勘合符的存在
  - 首辅借敬酒传递暗语：他知道沈时安在查，也知道旧档中有对太后不利的内容——但他不确定沈时安的立场
  - 卫将军抛出关键信息："不该看见的东西"——指遗诏伪造过程中的一个目击者
- **信息不对称**：
  - 沈时安已知：遗诏印文与勘合符印文同源
  - 太后开始怀疑：沈时安可能持有沈怀述遗留之物
  - 首辅不确定：沈时安是太后的棋子还是独立力量
  - 孙掌印知道：遗诏确系伪造——但他在等谁先出手
- **证物推进**：
  - 司礼监押印秘档：印文比对结果（私印非国玺）
  - 勘合符第一次出场：缺口处印文与遗诏印文吻合
  - 卫将军廊下碰面时暗示：两年前从沈怀述书房搜出的半枚勘合符，至今封存于兵部密档——"那半枚符还在。只是没人知道它和什么配对。"
- **谋局/反制**：
  - 太后设局（夜宴）试探沈时安底牌
  - 沈时安以女史身份遮掩（"年幼不知"）反制试探
  - 首辅以"阅了便罢"暗示警告——既保沈时安（不揭发她查档），又限制她行动
- **情绪节点**：
  - 印文比对发现吻合时：沈时安手指微颤——"是父亲的符。符上的印，与遗诏的印，同一枚。"
  - 夜宴试探：太后一句"你父亲可有留下什么"，沈时安答得不卑不亢——但放下酒杯时指尖入掌心
  - 卫将军廊下碰面：说完便走——"武将的脚步声在长廊上渐渐消了，沈时安低头看自己掌心的月牙形指甲印"
- **章末钩子**：沈时安回到值房，取出父亲密信再读。信末那句"切莫轻示于人"的下面——她用手指沾水涂抹——显出一行暗字："孙掌印知道真诏所在。但他不会说——除非有人先拿出证据。"
- **与 STORY-SPEC 对应**：
  - 证物锚点 §9（旧档印证押印疑点）
  - 信息不对称 §8（各方逐渐知道/怀疑的内容更新）
  - 谋局与反制 §9（设局试探 + 反制 + 新信息浮现）
  - 核心冲突 §4（术——用勘合符设局 vs 义——查真诏将暴露父罪）

### E03 - 旧邸书阁藏真诏，父罪新证裂人心

- **章节定位**：承——拆局受阻
- **核心事件**：
  - 沈时安以"为先父烧周年纸"为由出宫回旧邸。在书阁夹墙中找到真诏——先帝亲笔，立太子而非幼帝，诏末钤国玺真印。诏书夹层另有一纸——沈怀述当年的奏疏草稿，上疏内容为"遗诏押印与国玺不符"，但未发出即被截获
  - 真诏揭示两条信息：① 遗诏确系伪造；② 沈怀述两年前已知遗诏为伪，但他没有公开——因为公开将导致朝局动荡、太后/首辅/武将全面冲突。他选择了隐匿
  - 回宫途中，方尚仪在宫门等候——"首辅的人今天来调过旧档。我没给。但明天他们会带内阁的文书来。你只有今晚。"
  - 卫将军派人送来半句话："三日后登基。太后已命孙掌印明日将遗诏送去宗人府钤印——一旦钤印，真伪永不可辩。"
- **出场人物**：沈时安、方尚仪（宫门传信）、卫将军（间接——传话）、旧邸老仆（开书阁门——"这两年来，除了您，没人来过"）、萧太后（间接——登基倒计时催促）
- **权力冲突**：
  - 内部冲突激化：真诏在手 → 公开则父罪坐实（隐匿遗诏），不公开则伪诏立君。术与义的两难从理论变为现实
  - 外部时间压力：登基只剩三日，明日内阁将调走旧档，孙掌印将送遗诏钤印——三道时限压在同一天
  - 首辅的立场转换：派人调档意味着他也在查——但他查的是自保（消除自己的弹劾记录），而非真相
- **信息不对称**：
  - 沈时安已知：真诏位置 + 父曾隐匿 + 遗诏为伪 + 伪造者是孙掌印（奉太后意）
  - 太后已知：沈时安出宫——怀疑她去了旧邸
  - 首辅已知：旧档中有对自己不利的内容（当年弹劾沈怀述的奏疏）——他在销毁证据
  - 孙掌印已知：太后命明日钤印——但他不知道沈时安已经找到真诏
- **证物推进**：
  - 真诏（第 4 件证物）在旧邸书阁夹墙中被找到
  - 沈怀述奏疏草稿——证明他已知遗诏为伪但选择隐匿，构成"隐匿遗诏"的铁证
- **谋局/反制**：
  - 太后：加速登基程序（明日钤印）——时间压迫，不给人查证窗口
  - 首辅：调走旧档——以自保优先，不惜销毁可佐证真诏的证据链
  - 沈时安：必须在今晚做出决定——公开真诏还是隐匿到底
- **情绪节点**：
  - 找到真诏时：沈时安在书阁夹墙中摸到诏书卷轴——"她蹲在暗处，抱着那卷黄绫，额头抵在膝盖上。父亲的字。父亲的选择。"
  - 读到父奏疏时：沈时安终于理解父亲——"他不是懦弱。他是算过——公开之后死的人，比隐匿之后死的人多。"
  - 宫门口方尚仪递信：焦急但克制——"方尚仪说完便走，怕被人看见和自己站在一起的沈时安。"
- **章末钩子**：沈时安坐在值房内，面前摊着三样东西——真诏、父亲奏疏草稿、那半枚勘合符。烛火被窗缝漏进的夜风带得忽明忽暗。三日后登基，明日钤印，今夜必须决定。她拿起勘合符——符上铜边映出一线烛光。门外传来脚步声，不止一人。
- **与 STORY-SPEC 对应**：
  - 内部冲突 §7（忠义困境——术与义两难）
  - 证物锚点 §9（真诏 + 密信 / 奏疏草稿）
  - 结局方向 §11（代价已在此章埋下——公开真诏等于坐实父罪）
  - 核心创意 §4（"以家破换真相大白"的抉择在此章实质形成）

### E04 - 殿前对质验真印，两符合一揭伪造

- **章节定位**：转——证物翻盘
- **核心事件**：
  - 登基大典前一日清晨，沈时安请方尚仪代为通报——"婢子有先帝遗物，请呈太和殿。"
  - 殿前对质（三步翻盘）：
    - **设局**：沈时安将勘合符半枚呈给孙掌印鉴定。孙掌印验出印文与遗诏私印同源，太后当场反咬"此乃沈怀述伪造之印，意在乱政"
    - **拆局**：沈时安反问——"沈怀述已故两年，如何伪造两年前的遗诏？"徐首辅迟疑，未站队。萧太后催促"既无他证，将此婢拿下"
    - **反将**：卫将军携另半枚勘合符入殿——"此半枚勘合符，是两年前从沈怀述书房中搜出，存于兵部密档。两符合一，便是完整的司礼监掌印私章。"两符合一，印文完整——出自司礼监孙掌印之手。沈时安同时呈上真诏——先帝真迹，立太子，钤国玺真印。双证同时翻盘
  - 孙掌印伏罪——"是萧太后命老奴伪造。先帝驾崩那夜，太后便取走了国玺。"
  - 萧太后当殿失语——"她倚在御座扶手上，手中的那卷伪诏滑落玉阶。"
- **出场人物**：沈时安（翻盘者）、萧太后（被翻盘）、徐首辅（摇摆→最终承认真诏）、卫将军（关键援军——携另半枚勘合符到场）、孙掌印（伏罪）、方尚仪（旁听——在殿角记录）
- **权力冲突**：
  - 殿前对质本身即权力程序：按祖制，登基前一日可开"申议"——任何持证者可呈物证请三堂共验（内阁+司礼监+宗人府）
  - 三步递进中每一步都是信息释放：第一步抖出印文同源 → 第二步抖出时间矛盾 → 第三步抖出双证合璧
  - 卫将军的入殿时机是谋略安排——他在等太后反咬之后才现身，因为"只有她先否定沈怀述，你的铁证才能把她钉死。"
- **信息不对称（对质中逐步瓦解）**：
  - 对质前：太后以为沈时安只有半枚符（不足以定论）；沈时安已知卫将军有另半枚（但他们不能提前暴露）
  - 设局阶段：孙掌印验出印文同源 → 太后以为反咬沈怀述即可脱身
  - 拆局阶段：沈时安指出时间矛盾 → 太后仍寄望于"无其他证据"
  - 反将阶段：卫将军携另半枚符 + 真诏 → 信息差崩塌，伪造暴露
- **证物推进**：
  - 勘合符两半合一（第 5 件证物体系闭环）
  - 真诏首次公开（第 4 件证物入场）
  - 五证物全部兑现：勘合符（半枚→两半合一）、旧档（印文比对）、密信（指认真诏方向）、真诏（推翻伪诏）、勘合符另一半（证明私印来源）
- **谋局/反制**：
  - 三步翻盘全程符合 gudai-skill 要求：设局（主动出证）→ 拆局（指出逻辑矛盾）→ 反将（以彼之印反制彼身）
  - 不用巧合：勘合符两半在前文均有铺垫（沈时安持有半枚§E01，另半枚在兵部密档§E02卫将军暗示）
  - 不用武力：以物证和礼制程序翻盘
- **情绪节点**：
  - 沈时安入殿呈符：克制中的紧张——"她跪在殿中，双手捧符，指尖微颤但不洒。"
  - 太后反咬：沈时安低头不语——"她在等。等殿门外那个脚步声。"
  - 卫将军入殿：武将脚步沉重，铠甲轻响——"他单膝跪地，双手托起另半枚勘合符。两半铜符在孙掌印面前拼合——印文完整，分毫不差。"
  - 孙掌印伏罪：不是崩溃，是认了——"老奴只是掌印。印在谁手上，老奴便听谁的。"
  - 真诏展开：殿中百官跪伏——"先帝御笔。立太子。这份诏书，沈怀述守了两年。"
- **章末钩子**：殿中安静得只剩下烛花爆裂的声音。沈时安抬起头——徐首辅避开了她的目光，萧太后倚在御座上脸色灰白，孙掌印被押出殿外。卫将军把勘合符的两半并排放在她面前。"你的了。"他说。"但沈怀述隐匿遗诏——"徐首辅的声音从身后传来。

### E05 - 真相白沈门除籍，勘合焚烛灭殿空

- **章节定位**：合——成局与代价
- **核心事件**：
  - 真相大白之后，新帝（先帝太子）登基。萧太后迁居别宫，徐首辅自劾"失察之罪"留任，孙掌印赐死，卫将军自请解兵权归乡
  - 沈怀述案重审：①"私匿调兵文书"之罪撤销（勘合符合一证明那不是调兵文书，是先帝遗诏的证物）；②但"隐匿遗诏"之罪坐实——沈怀述两年来明知遗诏为伪而未举报
  - 沈时安以"持证翻盘有功"获免连带之罪，但沈家被除籍，沈时安不得留京。方尚仪送至宫门，交给她一只布包——里面是烧剩的半枚勘合符（已在殿前被孙掌印折辱时踩弯一角）
  - 卫将军在宫门外等她——"我送你一程。你父亲当年送过我，我欠他一程。"
- **出场人物**：沈时安、方尚仪（送别）、卫将军（送行——"坐到城外驿站，我便回去"）、萧太后（仅一句——迁居别宫时经过沈时安身边，没有看她）、徐首辅（间接——自劾奏疏中提到沈怀述"其心可悯、其行不可赦"）
- **权力冲突收束**：
  - 局势收束：新帝登基，遗诏归位，兵符交回——权力程序完整闭合
  - 势力重组：太后失势、首辅自保、武将解甲、内廷清洗——不是大团圆，是各付各的代价
  - 沈时安不在任何一方：她从头到尾都是夹缝中的棋子，翻盘之后仍然是棋子——只不过这次是她自己选择被放到棋盘外
- **信息不对称最终消解**：
  - 所有人都知道了真相——但真相的代价由沈家一家承担
  - 方尚仪最后才说："那年你父亲不是不敢公开——他是算过。公开之后，首辅会借机铲除太后一党，武将借机举兵，内廷借机清洗。死的人比现在多十倍。他选了最少的死人。"
- **证物终局**：
  - 真诏归位（收入宗人府秘档）
  - 勘合符焚毁——沈时安在离京前将其投入火盆："这是父亲的笔迹。不用留了。"
- **谋局/反制**：本章谋略退场，代价登场。这是 gudai-skill 结局公式的核心——"成局必失衡"。
- **情绪节点**：
  - 沈门除籍：沈时安在宗人府门口接过一纸公文——"上面写着父亲的名字，下面写着她的名字。两行字之间，隔了两年的沉默。"
  - 方尚仪送别：递布包时不说话——"她只是把布包塞进沈时安手里，然后捏了一下她的手。手很凉。"
  - 勘合符焚毁：沈时安蹲在宫门外，把半枚符投入火盆——"铜边烧黑了，印文渐渐看不清了。那是父亲最后的笔迹。现在没了。"
  - 卫将军送行：二人在马车上一路无话。到城外驿站时，卫将军说了唯一一句——"你父亲当年说过一句话。他说这个人以后会很累。他说的不是你。他说的是他自己。"
- **意象照应**：
  - 勘合符：半枚（E01）→ 比对（E02）→ 两符合一（E04）→ 焚毁（E05）
  - 烛：太和殿烛火晃动（E01）→ 值房烛忽明忽暗（E03）→ 殿前烛花爆裂（E04）→ 火盆中符燃烛灭（E05）
  - 旧邸书阁：被遗忘的空间（E03）→ 真相所在（E03）→ 提及但不再回去（E05——"旧邸要被收了，书阁里的灰，以后没人擦了"）
- **章末钩子**：马车出城。沈时安撩开车帘——远处皇城的角楼渐渐小了。她把布包放在膝上，包里的符已经凉了。车夫问："娘子，往哪儿走？""往南。一直往南。"
- **与 STORY-SPEC 对应**：
  - 结局方向 §11（局势收束 + 代价呈现 + 势力重组 + 意象照应 + 拒绝大团圆）
  - 核心创意 §4（"以家破换真相大白"完成）
  - 禁止项 §13（未用大团圆、未用全员和解、未用现代化总结陈词）
