## 3. 一句话梗概

先帝暴毙、遗诏押印存疑——失势旧臣之女持其父遗留的半枚勘合符，在太后、首辅与禁军三股势力夹缝中追溯旧案真相，最终翻盘证物须付出血亲代价。

## 4. 核心创意

一名失势旧臣之女，被迫卷入宫廷旧案。她手中仅有其父遗留的半枚勘合符（兵部调兵文书合验之印），缺口恰与先帝遗诏上的押印错位吻合。在信息不对称和权力夹缝中查清真相——遗诏确系伪造，真诏藏于旧邸。但翻盘需公开真诏，公开即意味着其父当年获罪时隐匿诏书之罪大白于天下，以父之名保全真相，以家破换真相大白。

**核心冲突**：术（用勘合符设局逼出伪造者）与义（揭真诏则父罪坐实，家族彻底覆灭）的两难。

## 7. 主要权力冲突

### 外部冲突（主线）

| 冲突维度 | 具体内容 |
|----------|----------|
| 势力对立 | 萧太后（掌遗诏）↔ 徐首辅（掌朝堂）↔ 卫将军（掌兵符）。三方围绕"遗诏真伪"角力：太后称遗诏立幼帝，首辅称遗诏押印有疑，武将称无真诏不调兵 |
| 关键资源 | 先帝遗诏（真/伪）、兵部勘合符（沈怀述遗留半枚）、司礼监押印秘档 |
| 时间压力 | 新帝登基大典在即（三日后），登基需遗诏。无真诏则登基不合法，三方各执一词无人敢先动 |
| 礼制制约 | 祖制：遗诏押印需三方（内阁+司礼监+宗人府），缺一不可；调兵需圣旨+虎符+勘合符三印合一 |

### 内部冲突（辅线）

| 冲突维度 | 具体内容 |
|----------|----------|
| 忠义困境 | 翻盘需公开真诏 → 真诏证明父曾隐匿 → 洗清"私匿调兵文书"的技术性污名，却换来"隐匿遗诏"的大罪。术与义无法两全 |
| 立场冲突 | 忠于父亲（私情）vs 忠于真相（公义）；接受太后庇护（生存）vs 揭露太后伪造（背叛） |

### 冲突递进

```
第1层（势力僵局）：遗诏押印存疑，三方对峙，新帝登基倒计时
    ↓
第2层（人心博弈）：沈时安借尚仪局身份查阅旧档，发现勘合符缺口与遗诏押印吻合
    ↓
第3层（大义拷问）：真诏所在被锁定——藏于沈家旧邸。公开则父罪坐实，不公开则伪诏立君
```

## 9. 谋局与反制方向

### 五个证物锚点

| 证物 | 出现章节 | 作用 |
|------|----------|------|
| 勘合符（半枚） | 第 1 章 | 主角持证，缺口与遗诏押印错位吻合——证明押印有问题 |
| 旧档（司礼监押印秘档） | 第 2 章 | 证明遗诏所用之印非国玺——乃司礼监私印 |
| 密信（沈怀述遗笔） | 第 3 章 | 指认真诏藏匿方向（旧邸书阁夹墙） |
| 真诏 | 第 4 章 | 先帝真迹，立太子而非幼帝——彻底推翻伪诏 |
| 勘合符另一半（兵部密档所存） | 第 4 章 | 太后误以为勘合符另一半已毁，未料兵部密档中仍存另一半——卫将军以此作为殿前翻盘的最后证物：两半勘合符合一，印文完整，证明私印来历 |

### 三步翻盘（第 4 章）

```
设局：沈时安将勘合符半枚呈给孙掌印，声称"先父遗物，请掌印鉴定"
拆局：孙掌印验出印文与遗诏私印同源 → 太后反咬"沈怀述伪造"
反将：沈时安请出真诏（旧邸夹墙）+ 卫将军携另半枚勘合符到场 → 两符合一 → 印文出自司礼监私章 → 伪造者乃孙掌印，奉太后意
```

## 10. 情绪基调

- 克制、冷峻：不以浓烈情绪渲染，以礼制动作为情绪载体（"移笏半寸""不过腰一揖""指尖入掌心"）
- 权力压迫感：宫墙/殿阶/烛影作为空间压迫意象；每一场对话都有第三者在场或潜在监听
- 礼制秩序下的暗流：表面是揖让进退，底层是杀机——如"此印，不当在阁下手中"，一句即翻脸
- 意象复现：勘合符（半枚→两半合一→焚毁）、烛（亮→摇曳→灭）、旧邸书阁（被遗忘的空间→真相所在）

## 11. 结局方向

- 局势收束：真诏现世，太后被迫退帘，徐首辅以"失察"自劾，卫将军以兵符换取沈时安免罪
- 代价呈现：沈怀述"隐匿遗诏"之罪坐实——沈时安洗清了"调兵文书"的小污名，换来"隐匿遗诏"的大罪名。沈家彻底除籍，沈时安以布衣之身离京
- 势力重组：新帝以太子身份登基，卫将军解兵权归乡，孙掌印赐死，太后迁居别宫
- 意象照应：沈时安离京前将半枚勘合符投入火盆——符上押印是父亲最后的笔迹，现在不需要了
- 拒绝大团圆：真相大白，但沈时安失去了父亲的名誉、家族的身份、宫中的立足之地。保住了真相，失去了一切
