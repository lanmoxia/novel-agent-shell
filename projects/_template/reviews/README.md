# reviews/

此目录由 Phase 5 script-review 生成审查报告。

## 预期文件

- `structure-review.md` — 结构审查
- `continuity-review.md` — 连续性审查
- `dialogue-review.md` — 对白审查
- `genre-review.md` — 题材符合度审查
- `final-review.md` — 最终审查报告（汇总以上四类）

## 禁止行为

- **禁止手动创建占位 final-review.md。** 占位文件在 stop-gate 中无效且会触发阻断。
- 审查报告由审查 Agent 生成，用户不可手动填写。
