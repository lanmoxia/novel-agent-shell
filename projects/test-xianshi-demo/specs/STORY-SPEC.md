## 三、核心创意

### 一句话梗概

一个 38 岁的 HR 在裁员名单上看到恩师的名字，签字则保住职位，不签则丢掉饭碗。

### 核心公式

```
可验证的困境 + 具体的压力源 + 不可兼得的选择 + 有代价的结局
```

### 创意说明

以职场中层管理者面临裁员任务为切口，展现规则与人情的两难困境。故事不依靠悬空设定或巧合，全部冲突源自可验证的现实压力（房贷、职业、人情债），最终以有代价的选择收束。

### 设计依据（摘自 outline-method.md 第四节）

- 职业：具体 + 中层（"某公司的财务主管"级）
- 年龄/位置：尴尬期（35 岁+）
- 性格标签：2-3 个具体特点（较真 + 流程控 + 怕冲突）
- 压力可量化：数字 + deadline

## 五、主要冲突

### 外部冲突（主线）

| 层级 | 冲突内容 |
|------|----------|
| **第 1 层：表面问题** | 签下裁员名单 → 保住职位；不签 → 自己上名单 |
| **第 2 层：关系撕裂** | 恩师是名单上的人，签了是背叛，不签是对公司规则的背叛 |
| **第 3 层：价值观拷问** | 规则正确 vs 人情正确 —— 两种"对"的碰撞 |

### 内部冲突（辅线）

- 道德困境：按流程执行 vs 欠恩师的人情债
- 价值观冲突：职场规则（流程即正义）vs 私人伦理（知恩图报）

### 冲突递进路径

```
第 1 章 → 裁员名单到手，恩师的名字赫然在列（表面冲击）
第 2-3 章 → 调查/拖延/试探，发现恩师被裁另有隐情（关系撕裂）
第 4 章 → 证据在手，必须做出最终决定（价值观拷问）
第 5 章 → 选择后的代价与余味（收束）
```

## 六、情绪基调

| 维度 | 设定 |
|------|------|
| 整体情绪 | 克制、收敛、不煽情 |
| 叙事距离 | 第三人称限知，贴近主角视线 |
| 语言风格 | 短句为主，白描，少形容词 |
| 节奏 | 紧凑，1-2 处现实压力峰值 |
| 结尾 | 留余味，非大团圆，用日常细节收束 |

### 依据（摘自 output-style.md）

- 语言克制朴素，短句为主，每句 ≤20 字
- 情绪收敛，避免说教和过度抒情
- 结尾留余味，用反差照应或静物对景

---

## 七、结局方向

**有代价的选择。**

- 主角选择了人情（或规则），付出了对应的代价。
- 代价不回避：保住良心则丢工作，守住规则则失去恩师的信任。
- 关系新平衡：用一次对话或行动收束，避免全员和解。
- 结尾用日常细节照应开头（如又一次打开招聘网站 / 又一次看到工位上的旧照片）。

### 禁忌
- 不假团圆
- 不过度解释
- 不说教
