## 原文事实

- 孔子东游，遇见两小儿争论。
- 两小儿争论太阳早晨和中午哪个时候离人更近。
- 小儿甲以太阳早晨大、中午小为证据。
- 小儿乙以早晨清凉、中午炎热为证据。
- 孔子不能决。
- 两小儿以“孰为汝多知乎？”形成反问。

## 教学点

- 分清两个观点。
- 理解两个证据。
- 理解“去”“及”“探汤”“孰”“汝”“知”等词句。
- 理解观察角度不同会产生不同判断。
- 理解学无止境和敢于质疑。

## Scene 映射记忆

- S01：孔子东游，见两小儿辩斗，问其故。
- S02：两小儿提出相反观点，只呈现观点层：日出近 / 日中近。
- S03：两小儿分别给出证据，只呈现证据层：大小证据 / 冷热证据。
- S04：孔子不能决。
- S05：两小儿反问并完成主题回扣，必须包含“孰为汝多知乎？”及白话“谁说你很有智慧呢？”。
