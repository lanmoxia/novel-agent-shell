## 2. 题材与风格锁定

- **黄河故道**：旧堤、废河槽、河工队旧事、旧河底泥
- **中式民俗恐怖**：不是鬼怪现身，而是旧物污染、禁忌传播、责任链
- **旧村禁忌**：命名/遮盖/观看/送返/传递五类
- **旧物污染**：泥封→返潮→湿泥→指印→裂缝→字迹渗入
- **禁忌传播**：通过观看、触碰、命名、记录扩散
- **责任链**：从拆墙发现者到知情者到证据持有者
- **文风**：写实、压抑、土腥、潮湿、克制；短句+停顿

## 3. 已确认核心设定

- **主角**：魏东来，28 岁，返乡翻修祖宅
- **故事入口**：拆灶房东墙发现夹墙 → 泥封旧木匣 → 河工工牌 + 三死结旧麻绳
- **核心旧物**：泥封旧木匣 + 河工工牌（赵满仓）
- **辅助旧物**：三死结旧麻绳、祖父旧账本
- **旧事来源**：三十年前雨夜加固旧堤，河工队 21 人出工 20 人回来；赵满仓被留在河下；救援绳被人打成死结；工牌和绳尾被魏东来的祖父封进木匣砌入灶房夹墙
- **村庄禁忌**：工牌名字不能念全、木匣泥封不能打开、工牌不能留在活人住的屋里
- **责任链传播**：拆墙（魏东来）→ 触碰工牌（魏东来）→ 念出名字（魏东来）→ 追问老人（魏东来）→ 老人被迫说出（陈伯/老周头）→ 远亲得知（刘婶）→ 名字进入笔记本记录
- **结局方向**：工牌送祠，名字已进入记录无法收回；灶房墙根湿泥仍在渗；笔记本名字持续出现；不追查到具体凶手；不可逆

## 5. 人物记忆

### 魏东来（主角）
- 身份：28 岁返乡青年，省城装修工程从业者
- 动机：翻修老宅（不是查旧事）
- 知道什么：不知道祖父是记工员，不知道夹墙藏匣，不知道赵满仓
- 不能写成：侦探、推理者、开挂者、通灵者、完美良心

### 陈伯（守祠老人）
- 身份：七十多岁，旧祠堂看守人，河工队幸存成员
- 功能：旧事知情人，从 E01 阻止到 E04 被迫说出
- 知道什么：知道木匣被砌进墙里但不知具体哪户
- 不能写成：神棍、旁白解说者、一次性讲完旧事、全知角色

### 老周头（村口老人）
- 身份：当年河工队后勤
- 功能："河下有人"半句话源头，禁念名字，逐级传送信息
- 知道什么：知道少了一人但不知道细节
- 不能写成：知情者、解说者

### 表叔老魏
- 身份：六十多岁，魏东来表叔
- 功能：提供半截信息——"你爷爷是记工员""册上的名没少，是划掉了"
- 知道什么：知道宅子是河工歇脚处，不知道墙里砌了东西
- 不能写成：全知者

### 刘婶
- 身份：赵满仓的远亲，镇上居民
- 功能：旧事收束——提供赵满仓人物信息，提出"他知不知道打完会怎样"
- 知道什么：知道人没回来但不知道原因（E04 之前）
- 不能写成：复仇者、指控者

### 村里少年
- 身份：村口碰见的孩子
- 功能："看见但看不懂"视角，E01 一句"我爷爷说那房子以前是河工歇脚的"
- 不能写成：后续主角、解说者

### 村主任
- 身份：仅 E02 电话出现一次
- 功能：压事者，叫停翻修
- 不能写成：反派、知情人、线索提供者
