## 4. Phase 5 P2 问题

Phase 5 记录的 P2 为：

1. 教材版本、目标年级、最终时长和原文采用版本仍需正式化前确认。
2. 草案局部偏教学流程卡，观看张力偏弱。
3. 司马光判断瞬间需要更自然、更儿童化。
4. 群儿调度需压缩，避免单镜头互动人物超过 3 个。
5. 坠瓮、水面、击瓮和水迸动作需安全化表达。
6. S05 学习闭环略直白，需在 Phase 6 柔化。

## 5. 本轮处理内容

本轮已处理不依赖用户确认的 P2：

- S01 增加竹圈滚向瓮边、瓮旁湿滑和司马光提醒，强化可观看性和轻微风险预示。
- S02 改为用竹圈落地、水声和孩子反应表现危机，避免惊悚细节。
- S03 让司马光看水声、瓮壁和石块，用 `水出去了，人就能出来。` 表达短促判断。
- S04 使用 `砸瓮，让水走！`，让判断更儿童化，不做成人式长讲解。
- 每场均控制近景主要互动人物不超过 3 个，群儿反应用远景和声音处理。
- S04 安全化击瓮表现，不写碎片飞溅、武打动作或暴力爽点。
- S05 改为从小儿甲的事件复述自然回扣原文顺序，减少硬总结感。
