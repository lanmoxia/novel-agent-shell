# 审查评分与锁稿标准

## 统一评分

每类审查使用 0-5 分：

| 分数 | 含义 |
|------|------|
| 5 | 完全达标，无明显问题 |
| 4 | 达标，仅有轻微优化点 |
| 3 | 基本达标，有中等问题但不破坏整体 |
| 2 | 不达标，存在影响阅读或逻辑的问题 |
| 1 | 严重不达标，多处核心问题 |
| 0 | 无法判断或完全失效 |

## 锁稿最低线

- 不得存在未修复的严重问题
- 任一审查维度低于 3 分，不得锁稿
- 题材禁忌触发且未修复，不得锁稿
- final-review.md 必须明确结论为“通过”

## 处理建议

| 问题类型 | 建议 |
|----------|------|
| 单句表达、错名、格式错误 | 局部修补 |
| 人物口吻、局部连续性、单章节奏 | 局部重写或整场重写候选 |
| 章节功能失败、信息释放错误 | 整章/整集重写候选 |
| 核心真相、结局、主线结构错误 | 回到 Phase 2/3 重新规划 |
