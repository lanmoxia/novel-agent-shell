# Feedback 反馈记录目录

此目录用于存储用户反馈记录和进化提案，由 feedback-observer / evolution-runner 写入。

## 文件命名规则

```
YYYYMMDD-HHMMSS-feedback.md
```

## 索引文件

`feedback-index.md` 维护所有反馈记录的索引表。

## 进化提案

`proposals/` 存放规则进化提案。提案只记录建议，不自动修改 `ORCHESTRATION.md`、Skill、Agent、Hook 或项目记忆。

提案命名：

```
YYYYMMDD-HHMMSS-proposal.md
```
