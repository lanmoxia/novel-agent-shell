# 结构规则

## 1. 生成前必须先拆原文

不得直接写剧本。

生成剧本前，必须先完成：

1. 原文标题
2. 原文出处
3. 核心人物
4. 事件起因
5. 事件经过
6. 事件结果
7. 关键原文句
8. 白话释义
9. 核心主题
10. 学生学习难点

## 2. 默认结构

默认使用：

- T-WY001：情境复演型

标准结构：

```txt
题目钩子
→ 古风场景建立
→ 原文事件触发
→ 人物冲突出现
→ 关键人物介入
→ 观点 / 行为展开
→ 证据 / 后果呈现
→ 结果 / 反转
→ 原文回扣
→ 白话释义
→ 主题收束
```

## 3. 60 秒结构

适合短视频快速传播。

```txt
0-3s：题目钩子
3-10s：古风场景建立
10-25s：事件触发
25-40s：冲突展开
40-50s：结果 / 反转
50-60s：原文回扣 + 主题
```

## 4. 90 秒结构

适合兼顾剧情和教学。

```txt
0-5s：题目钩子
5-15s：场景建立
15-30s：事件触发
30-55s：冲突推进
55-70s：结果 / 反转
70-85s：原文回扣
85-90s：主题总结
```

## 5. 180 秒结构

适合完整讲解型视频。

```txt
0-8s：题目导入
8-25s：古风场景建立
25-55s：事件触发
55-100s：观点 / 行为展开
100-130s：结果 / 反转
130-165s：原文逐句回扣
165-180s：主题收束
```

## 6. Scene 规则

每个 Scene 必须包含：

1. 时长
2. 场景
3. 人物
4. 对应原文
5. 剧情功能
6. 知识功能
7. 画面描述
8. 动作
9. 对白 / 旁白
10. 字幕

## 7. 原文映射规则

每个剧情段落必须绑定：

1. 对应原文
2. 白话意思
3. 剧情表现
4. 学习功能

不得出现“有剧情但无原文对应”的段落。

## 8. 冲突设计规则

冲突必须来自原文。

允许：

- 观点冲突
- 行为错误
- 判断困难
- 误解
- 问答
- 观察差异
- 结果反转

禁止：

- 编造打斗
- 编造爱情
- 编造权谋
- 编造仇恨
- 编造神怪冲突

## 9. 结尾规则

结尾必须完成至少两项：

1. 原文关键句回扣
2. 白话释义
3. 文章主题
4. 考点提示
5. 背诵顺序提示
