# 禁忌规则

## 1. 内容禁忌

不得出现：

1. 脱离教材原文乱编。
2. 改变核心人物关系。
3. 改变原文结果。
4. 加入爱情线。
5. 加入权谋线。
6. 加入仙侠、法术、神怪元素。
7. 把儿童角色写得成人化。
8. 使用网络梗。
9. 用现代价值观强行改写古文。
10. 为了戏剧冲突牺牲知识准确性。

## 2. 结构禁忌

不得出现：

1. 直接跳过原文拆解。
2. 有剧情但无原文映射。
3. 有原文但无白话解释。
4. 结尾没有主题总结。
5. 全程只靠旁白讲解。
6. 原文顺序被打乱且不说明。
7. 前后人物关系不一致。
8. 冲突不是来自原文。

## 3. 画面禁忌

不得出现：

1. 多人打斗。
2. 复杂群演。
3. 频繁换场景。
4. 大幅度身体接触。
5. 高速追逐。
6. 过度华丽服饰。
7. 仙侠滤镜。
8. 游戏 CG 风。
9. 现代物品穿帮。
10. 角色脸频繁变化。

## 4. 字幕禁忌

不得出现：

1. 整段原文铺满屏幕。
2. 只有原文没有白话。
3. 解释错原文。
4. 字幕和剧情脱节。
5. 用过长句子解释。
6. 字幕层级混乱。
7. 原文字幕被口语化改写。

## 5. 教学禁忌

不得出现：

1. 错误翻译。
2. 错误主题。
3. 错误人物身份。
4. 错误事件顺序。
5. 错误考点归纳。
6. 把争辩型文章讲成单方正确。
7. 把寓言型文章讲成单纯笑话。
