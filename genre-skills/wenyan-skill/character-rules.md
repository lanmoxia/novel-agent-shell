# 人物规则

## 1. 总原则

人物必须服务原文和教学目标。

不得为了戏剧性强行增加无关人物。

## 2. 人物数量

推荐每集主要人物数量：

- 2 人：最稳定
- 3 人：推荐上限
- 4 人：需要谨慎
- 5 人以上：不建议

单镜头主要互动人物不得超过 3 个。

## 3. 常见人物类型

### 3.1 原文主角

功能：

- 承担核心事件
- 推动剧情
- 展示文章主题

要求：

- 性格符合原文
- 行动符合原文
- 不能被改写成无关人设

### 3.2 冲突对象

功能：

- 提出不同观点
- 制造问题
- 引发争辩
- 衬托主角

要求：

- 冲突必须来自原文
- 不能恶意丑化
- 不能加入无关矛盾

### 3.3 权威人物

常见：

- 孔子
- 老师
- 长者
- 官员
- 父亲
- 先生

功能：

- 评判
- 提问
- 引导
- 见证反转

要求：

- 不能神化
- 不能强行正确
- 如果原文写“不知”或“不能决”，必须保留

### 3.4 儿童角色

功能：

- 提问
- 争辩
- 观察
- 犯错
- 展示童真和智慧

要求：

- 语言简单
- 行动清楚
- 不成人化
- 不油腻
- 不说网络梗

## 4. 角色一致性

每个主要角色必须固定：

1. 年龄
2. 发型
3. 服装颜色
4. 身高比例
5. 气质
6. 道具
7. 面部特征

## 5. 典型角色配置

### 争辩型文本

```txt
两个争辩者
+ 一个评判者
+ 可选旁观者
```

### 寓言型文本

```txt
犯错者
+ 旁观者
+ 结果见证者
```

### 机智型文本

```txt
问题制造者
+ 少年主角
+ 旁观众人
```

### 劝诫型文本

```txt
被劝者
+ 劝诫者
+ 事件结果
```

## 6. 禁止人物设计

禁止：

1. 无关反派
2. 爱情对象
3. 玄幻导师
4. 神仙
5. 妖怪
6. 皇帝权谋线
7. 现代穿越者
8. 搞笑网红式旁白角色
