# 题材定位：文言文教辅真人古风短剧

## 1. 题材本质

本题材的本质是：

```txt
教材文言文
→ 原文事件拆解
→ 古风真人情境复演
→ 人物冲突 / 行动 / 问答
→ 原文字幕回扣
→ 白话释义
→ 主题总结
```

它不是单纯讲故事，也不是单纯讲课。

它要同时满足：

1. 有剧情。
2. 有知识点。
3. 有原文映射。
4. 有白话解释。
5. 有视频生成可行性。

## 2. 目标用户

主要面向：

- 小学生
- 初中生
- 家长
- 教辅内容账号
- 老师辅助讲解场景

## 3. 学习目标

常见学习目标包括：

1. 看懂文言文事件。
2. 理清人物关系。
3. 理解关键文言句。
4. 记住文章结构。
5. 明确文章主题。
6. 辅助背诵。
7. 辅助考试答题。

## 4. 适用文本特征

适合处理的文本通常具备：

1. 人物少。
2. 事件清楚。
3. 情节短。
4. 有问答。
5. 有冲突。
6. 有判断。
7. 有反转。
8. 有明显道理或主题。

## 5. 不适用文本特征

不适合处理的文本通常是：

1. 无人物。
2. 无事件。
3. 无冲突。
4. 纯抒情。
5. 纯写景。
6. 篇幅过长。
7. 论述过抽象。
8. 需要大量背景知识才能理解。

## 6. 内容底线

生成内容必须优先保证：

1. 原文不被误读。
2. 人物关系不被改错。
3. 核心事件不被改写。
4. 主题不被扭曲。
5. 不能为了戏剧性牺牲教学准确性。
