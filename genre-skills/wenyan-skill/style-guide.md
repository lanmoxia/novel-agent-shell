# 风格规则

## 1. 总体风格

本 Skill 的默认风格为：

```txt
真人古风
+ 教材感
+ 轻剧情
+ 清晰讲解
+ 适合学生理解
```

风格不能走偏。

## 2. 画面气质

推荐画面气质：

1. 写实古风。
2. 干净明亮。
3. 朴素自然。
4. 有教材插图感。
5. 有短剧可看性。
6. 不追求过度电影化。

## 3. 古风程度

允许：

- 古代服饰
- 古道
- 村口
- 河边
- 树下
- 学堂
- 庭院
- 田野
- 简单车马
- 竹简、书卷、木牌等道具

禁止：

- 仙侠化
- 玄幻化
- 神怪化
- 权谋化
- 宫斗化
- 过度华丽化
- 网游 CG 风
- 影楼风

## 4. 教材感

必须保留一定教材感。

教材感来源包括：

1. 明确篇目标题。
2. 原文字幕。
3. 白话解释。
4. 主题总结。
5. 原文与剧情逐句对应。
6. 结尾有学习闭环。

## 5. 短剧感

不能完全变成微课。

短剧感来源包括：

1. 开头有问题。
2. 人物有冲突。
3. 场景有行动。
4. 角色有问答。
5. 结尾有反转或收束。

## 6. 视觉复杂度

为了适配 AI 视频生成，必须控制复杂度。

优先：

- 中景对白
- 近景表情
- 远景建立环境
- 空镜辅助解释
- 侧脸
- 背影
- 慢动作

避免：

- 快速追逐
- 多人打斗
- 复杂群演
- 频繁换场
- 大幅肢体接触
- 多角色同时说话

## 7. 色调建议

推荐：

- 自然光
- 暖色晨光
- 柔和午后光
- 清淡古风色调
- 真实户外质感

避免：

- 高饱和网游感
- 强滤镜仙侠感
- 过暗悬疑感
- 过度磨皮
- 过度梦幻
