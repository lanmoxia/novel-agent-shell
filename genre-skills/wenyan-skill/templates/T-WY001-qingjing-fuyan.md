# T-WY001：情境复演型模板

## 1. 模板定位

本模板用于将故事型文言文改编为真人古风教辅短剧。

适用于有明确人物、事件、冲突、问答、判断或反转的文言文。

## 2. 核心结构

```txt
题目钩子
→ 古风场景建立
→ 原文事件触发
→ 人物冲突出现
→ 关键人物介入
→ 观点 / 行为展开
→ 证据 / 后果呈现
→ 结果 / 反转
→ 原文回扣
→ 白话释义
→ 主题收束
```

## 3. 适用篇目

- 《两小儿辩日》
- 《王戎不取道旁李》
- 《司马光》
- 《守株待兔》
- 《揠苗助长》
- 《刻舟求剑》
- 《曹冲称象》
- 《杨氏之子》

## 4. 60 秒结构

| 时间 | 内容 | 功能 |
|---:|---|---|
| 0-3s | 题目钩子 | 告诉观众本集讲什么 |
| 3-10s | 古风场景建立 | 把原文第一句转成画面 |
| 10-25s | 事件触发 | 引出人物和冲突 |
| 25-40s | 冲突展开 | 呈现观点、行为或错误 |
| 40-50s | 结果 / 反转 | 对应原文结果 |
| 50-60s | 原文回扣 + 主题 | 完成学习闭环 |

## 5. 90 秒结构

| 时间 | 内容 | 功能 |
|---:|---|---|
| 0-5s | 题目钩子 | 建立篇目与问题 |
| 5-15s | 场景建立 | 还原故事背景 |
| 15-30s | 事件触发 | 进入原文事件 |
| 30-55s | 冲突推进 | 呈现关键句意 |
| 55-70s | 结果 / 反转 | 还原文本结果 |
| 70-85s | 原文回扣 | 原文 + 白话解释 |
| 85-90s | 主题总结 | 输出文章道理 |

## 6. 180 秒结构

| 时间 | 内容 | 功能 |
|---:|---|---|
| 0-8s | 题目导入 | 建立篇目 |
| 8-25s | 古风场景建立 | 还原故事背景 |
| 25-55s | 事件触发 | 引出核心事件 |
| 55-100s | 观点 / 行为展开 | 展示关键逻辑 |
| 100-130s | 结果 / 反转 | 对应原文结果 |
| 130-165s | 原文逐句回扣 | 原文 + 白话解释 |
| 165-180s | 主题收束 | 总结道理或考点 |

## 7. 人物配置

基础公式：

```txt
原文主角
+ 冲突对象
+ 评判者 / 旁观者 / 引导者
```

争辩型文本公式：

```txt
两个争辩者
+ 一个评判者
+ 可选旁观者
```

寓言型文本公式：

```txt
犯错者
+ 旁观者
+ 后果呈现
```

机智型文本公式：

```txt
问题制造者
+ 少年主角
+ 旁观众人
```

## 8. 场景配置

优先使用：

- 古道
- 村口
- 河边
- 树下
- 学堂
- 田野
- 庭院
- 市集

一集最多使用 2-3 个主场景。

## 9. 知识植入优先级

优先级如下：

1. 角色行动表现原文。
2. 角色对白解释句意。
3. 字幕展示原文和白话。
4. 旁白总结主题。

不得全靠旁白讲解。

## 10. 结尾闭环

结尾必须至少包含以下两项：

1. 原文关键句回扣
2. 白话释义
3. 文章主题
4. 考点提示
5. 背诵顺序提示

## 11. 生成约束

1. 不得脱离原文事件链。
2. 不得加入无关剧情。
3. 不得把古文改成仙侠故事。
4. 不得把教辅视频写成纯短剧。
5. 不得让角色说现代网络梗。
6. 每个 Scene 必须对应原文或教学点。
