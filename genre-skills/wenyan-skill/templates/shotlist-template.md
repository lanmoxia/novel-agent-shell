# 分镜输出模板

## 1. 分镜总表

| 镜头编号 | 时长 | 景别 | 场景 | 人物 | 动作 | 对应原文 | 知识功能 | 生成方式 | 风险 |
|---|---:|---|---|---|---|---|---|---|---|
| S01 |  |  |  |  |  |  |  |  |  |

## 2. 字段说明

### 镜头编号

使用：

```txt
S01
S02
S03
```

### 时长

建议单镜头：

- 3 秒
- 4 秒
- 5 秒
- 6 秒
- 8 秒

避免单镜头超过 10 秒。

### 景别

可选：

- 远景
- 中景
- 近景
- 特写
- 空镜
- 背影
- 侧脸

### 生成方式

可选：

- 文生视频
- 图生视频
- 图片静帧
- 空镜生成

### 风险

可选：

- 低
- 中
- 高

## 3. 分镜规则

1. 单镜头只表达一个动作。
2. 单镜头只承载一个知识点。
3. 单镜头主要互动人物不超过 3 个。
4. 有复杂对白时优先用中景。
5. 解释自然现象时可用空镜。
6. 角色脸容易不稳定时，用侧脸、背影或远景。
7. 每个镜头都要绑定原文或教学功能。

## 4. 风险判断

### 低风险

- 空镜
- 单人站立
- 单人行走
- 中景对白
- 近景表情
- 背影

### 中风险

- 两人争辩
- 三人同框
- 指向远处
- 递交道具
- 简单转身

### 高风险

- 推搡
- 打斗
- 追逐
- 多人复杂互动
- 儿童快速运动
- 群演同时行动
