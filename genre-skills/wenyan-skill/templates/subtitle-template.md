# 字幕输出模板

## 1. 字幕总表

| 字幕编号 | 字幕类型 | 内容 | 对应画面 | 出现位置 | 功能 |
|---|---|---|---|---|---|
| SUB01 | 原文字幕 |  |  |  |  |
| SUB02 | 白话字幕 |  |  |  |  |
| SUB03 | 对白字幕 |  |  |  |  |

## 2. 字幕类型

### 原文字幕

用于展示教材原文。

要求：

1. 忠于原文。
2. 逐句出现。
3. 不得改写。
4. 不得整段堆满屏幕。

### 白话字幕

用于解释文言句意。

要求：

1. 简单。
2. 准确。
3. 口语化。
4. 学生能看懂。

### 对白字幕

用于角色说话。

要求：

1. 简短。
2. 清楚。
3. 不抢画面。
4. 与角色动作一致。

## 3. 推荐字幕格式

```txt
第一行：原文
第二行：白话解释
```

示例：

```txt
孔子东游，见两小儿辩斗。
孔子向东方游历，看见两个孩子正在争论。
```

## 4. 字幕位置

推荐：

- 原文字幕：画面中下方
- 白话字幕：原文字幕下方
- 对白字幕：底部居中
- 标题字幕：画面中央或上方

## 5. 字幕禁忌

1. 不要整段原文铺满屏幕。
2. 不要只有原文没有解释。
3. 不要解释错原文。
4. 不要字幕和剧情脱节。
5. 不要一句字幕超过 20 个字太多。
6. 不要同时出现太多层字幕。
