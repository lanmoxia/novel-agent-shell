# 台词规则

## 1. 台词类型

本 Skill 中有四类文本：

1. 角色对白
2. 旁白
3. 原文字幕
4. 白话释义字幕

四类文本必须区分，不能混用。

## 2. 角色对白

角色对白用于推进剧情和表达人物观点。

要求：

1. 简短。
2. 清楚。
3. 适合学生理解。
4. 符合古风情境。
5. 不要过度文绉绉。
6. 不要现代网络化。

示例：

```txt
小儿甲：我看太阳刚升起时更大，当然离我们更近！
小儿乙：不对！中午太阳晒得人发烫，那才是离我们近！
```

## 3. 旁白

旁白用于补充背景和总结。

要求：

1. 不抢剧情。
2. 不整段讲课。
3. 只在必要时使用。
4. 优先解释原文背景和主题。

示例：

```txt
旁白：孔子向东方游历，路上遇见两个孩子正在争辩。
```

## 4. 原文字幕

原文字幕必须忠于教材原文。

要求：

1. 逐句呈现。
2. 不得改写。
3. 不得乱加标点导致误读。
4. 必须和剧情画面对应。

示例：

```txt
孔子东游，见两小儿辩斗。
```

## 5. 白话释义字幕

白话释义用于解释原文。

要求：

1. 口语化。
2. 学生能看懂。
3. 不学术化。
4. 不成人化。
5. 不能解释错意思。

示例：

```txt
孔子向东方游历，看见两个孩子正在争论。
```

## 6. 字幕层级

推荐格式：

```txt
第一行：原文
第二行：白话解释
```

示例：

```txt
我以日始出时去人近，而日中时远也
我认为太阳刚升起时离人近，中午时离人远。
```

## 7. 禁止台词

禁止：

1. 网络梗。
2. 现代流行语。
3. 成人化调侃。
4. 油腻对白。
5. 无意义口号。
6. 与原文无关的鸡汤。
7. 太长的讲课式对白。
8. 角色突然说完整考点解析。
