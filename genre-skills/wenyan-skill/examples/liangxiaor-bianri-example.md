# 示例：《两小儿辩日》结构拆解

## 1. 篇目信息

- 标题：《两小儿辩日》
- 类型：文言文故事
- 模板：T-WY001 情境复演型
- 核心人物：孔子、两个小儿
- 核心事件：两个孩子争论太阳早晨和中午哪个时候离人更近
- 核心主题：学无止境，要敢于质疑，也要尊重观察和思考

## 2. 原文结构拆解

| 原文句子 | 白话意思 | 剧情功能 | 学习功能 |
|---|---|---|---|
| 孔子东游，见两小儿辩斗，问其故。 | 孔子向东方游历，看见两个孩子争论，便问原因。 | 建立背景，引出冲突 | 理解人物和事件起点 |
| 一儿曰：我以日始出时去人近，而日中时远也。 | 一个孩子说：我认为太阳刚升起时离人近，中午时离人远。 | 提出观点 A | 理解第一方观点 |
| 一儿曰：我以日初出远，而日中时近也。 | 另一个孩子说：我认为太阳刚升起时远，中午时近。 | 提出观点 B | 理解第二方观点 |
| 一儿曰：日初出大如车盖，及日中则如盘盂。 | 一个孩子说：太阳刚升起时大得像车盖，到中午像盘子。 | 提供证据 A | 理解“大远小近”的判断 |
| 一儿曰：日初出沧沧凉凉，及其日中如探汤。 | 另一个孩子说：太阳刚升起时清凉，到中午热得像手伸进热水。 | 提供证据 B | 理解“冷热感受”的判断 |
| 孔子不能决也。 | 孔子也不能判断。 | 形成反转 | 理解圣人也有不知道的事 |
| 两小儿笑曰：孰为汝多知乎？ | 两个孩子笑着说：谁说你很有智慧呢？ | 主题收束 | 理解质疑精神和学无止境 |

## 3. 剧情结构

```txt
题目钩子
→ 孔子东游
→ 遇见两小儿争辩
→ 孔子询问原因
→ 小儿甲提出观点
→ 小儿乙反驳
→ 双方分别举证
→ 孔子无法判断
→ 两小儿发问
→ 原文回扣与主题总结
```

## 4. 可复用规则

1. “孔子东游”转化为古道行走画面。
2. “两小儿辩斗”转化为两个孩子在路边争论。
3. 两个观点必须成对出现。
4. 两个证据必须成对出现。
5. “孔子不能决”必须保留，不能改成孔子强行讲正确答案。
6. 结尾必须回到“学无止境 / 敢于质疑 / 观察角度不同”主题。
7. 不要添加打斗、追逐、穿越、仙侠内容。
