# 人物小传

## 主要角色

**柳映雪（主角）** 19岁，江南柳氏商户次女，因长姐病亡替嫁入三皇子府。性情爽朗，不谙深宫机巧，却聪慧善察。婚后才知嫁妆中的"祖传玉佩"是先帝私印，被迫卷入储位之争。她在三日内分别查证三方底细，确认三皇子德行可堪，于册封大典献印押诏。代价是一生困于宫墙，与萧瑾言虽为夫妻却形同君臣，终生未育，晚年抄经度日。她说："先帝将印托付柳家，便是要我们识忠奸。臣妾选了陛下，是因陛下确实可堪大任，而非儿女私情。"

## 重要配角

**萧瑾言（三皇子）** 26岁，文韬武略，掌南军调度，谋储位。知私印在柳家，特意求娶柳映月，柳映月病亡后仍娶柳映雪。他对柳映雪坦言："此物于本王而言，关乎社稷。"又说："先帝将印托付柳家，便是要你们识忠奸。若你认为本王不配，本王不强求。"继位后封柳映雪为贵妃，后宫只她一人，但两人始终隔着"利用"的阴影，客气疏离。他曾问："若有来生，朕不做皇子，你不做王妃，可好？"

**太后** 60岁，掌内廷司礼监，四皇子生母，欲改立幼子为储。她在先帝病重时曾多次召内侍入寝宫，有篡改遗诏之嫌。因无真印，她仿制假印，在册封大典上试图蒙混，被顾太傅当场验出。她曾遣人诱柳映雪："王妃若肯献印于太后，太后自会保王妃一生富贵。"新君继位后，她被软禁慈宁宫。代表权力场中不择手段的野心家。

**萧瑾安（大皇子）** 30岁，掌北军兵符，武将出身，与三皇子争储。他曾在边关私调兵马，有前科。他对萧瑾言言道："三弟若能寻得先帝私印，不妨交兄长保管，免得引火烧身。"虽掌兵权，但无遗诏正名，不敢轻举妄动。新君继位后，他被削去兵符，贬为郡王。代表有兵无谋的武夫，空有武力却不得人心。

**顾太傅** 70岁，三朝元老，掌礼部，手握先帝遗诏原件。遗诏书"立三皇子萧瑾言为储"，笔迹是先帝亲笔，但押印处空白。他对柳映雪言："先帝临终前，亲书此诏，却故意不押印。先帝言'朕将私印托付柳家，若柳家长女认为瑾言可堪大任，自会献印'。"他在册封大典上验出太后的假印，宣读遗诏让三皇子继位。新君继位后，他功成身退，辞官归乡。代表忠臣的智慧——既守先帝遗愿，又让继承人经受考验。

## 其他角色

**柳映月** 柳映雪的长姐，已故。本是三皇子原配人选，婚前病亡。她生前贴身保管先帝私印（父亲对外称"祖传玉佩"），临终前交给妹妹。她的死让柳映雪替嫁，也让私印落入柳映雪手中。

**柳父** 柳氏商户家主，已故。三年前先帝南巡曾住柳家，离开时将私印托付给他，并留密信："此印交予柳家长女，由她决定交予何人。朕信她能识忠奸。"他临终前嘱咐长女贴身保管，却没来得及说明真相。
