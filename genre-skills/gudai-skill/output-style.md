# 写作风格总则
- 文白相济：叙述端庄、对白简练含锋；礼制称谓准确
- 章法有"回目"感：小标题点题、段末留悬，夹叙夹议适量
- 多线并行：阵营/派系错位推进，以信息不对称造悬念
- 节奏主轴：谋略→揭示→翻盘；每步抖出硬信息
- 计谋与人心并重：术与义相互牵制
- 意象复现：印、剑、笏、烛、雨等物反复照应
- 结局见代价：成局必失衡，胜利伴牺牲
- 融会大纲：章节大纲为分类信息，非叙述顺序；创作时需按时间推进重组信息，切忌逐块照搬；若大纲分"权力僵局、公主处境、三方角力"，叙述应为"驾崩→朝争→试探"的事件流，背景信息自然穿插其中

## 具体行为
- 写对话时：礼称准确、含而不露；关键句留半步，如"此印，不当在阁下手中"
- 写动作时：仪节与细部支撑身份，如"不过腰一揖""移笏半寸"
- 写转折时：以诏、印、供、账、图为证具翻盘；或以权力程序反制
- 写爽点时：三步递进"设局—拆局—反将"，每步抖出新信息
- 写虐点时：以忠义/骨肉决断呈现，如"一纸换一族"
- 处理伏笔时：前文以典章细节轻触（文书缺口、押印错行），回收以旁证/证人/旧物
- 每场结尾：留钩子在权术动作上，如"灯一灭""印落玉阶""内侍入报未尽"
