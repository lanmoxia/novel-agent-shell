# 结构规则：从旧村日常推向不可逆后果

## 核心结构

默认结构：日常旧村 → 异常旧物 → 村民避讳 → 旧事露头 → 禁忌被破 → 后果渗出 → 不可逆收束。

这是推进逻辑，不是固定章节名。无论写短篇、单集短剧、多集短剧还是长篇小说，都必须保留这条递进链。

## 第一阶段：日常旧村

开场先建立真实生活。可用入口：清淤、修堤、翻旧宅、村口闲坐、饭桌吃饭、老槐树下聊天、办白事、雨后排水、河工收工、祠堂清理。

目标是让读者相信这是一个真实村庄。禁止一开场就鬼脸、惨叫、血字、神秘人讲传说、大段旁白解释禁忌。

## 第二阶段：异常旧物

异常应从旧物进入。旧物出现必须写清：谁发现、从哪里出来、怎样被拿出来、初始状态是什么、谁第一个碰、谁第一个变脸、旧物暂时放在哪里。

旧物不能凭空出现，也不能只在吓人时出现。旧物类型应根据项目选择，不默认复用固定组合。

## 第三阶段：村民避讳

旧物出现后，不要立刻解释真相。更好的反应：有人嘴硬，有人打岔，有人让盖上，有人让孩子走开，有人说“别在这儿说”，有人看向老人，有人转身离开，有人提醒“按老规矩办”。

避讳要制造群体压力：村里不是没人知道，而是没人愿意第一个说。

## 第四阶段：旧事露头

旧事只能露出一部分。可用方式：老人半句话、河工传闻、旧名册缺页、旧账本错行、旧照片被剪掉的人、村志涂改、祠堂偏间、废渡口旧碑、旧地名牌、出工牌背面刻痕。

每次只给一个新信息。禁止一次讲清旧事起因、禁忌规则、传播机制、破解方法和最终真相。

## 第五阶段：禁忌被破

破禁不能只是“有人作死”。破禁要有现实动机：贪便宜、嘴硬要面子、急着证明自己没错、为了救人、为了查父辈旧事、为了隐瞒旧账、为了推卸责任、因工作要求不得不碰。

破禁后不要马上爆炸式惩罚。先出现小偏差：旧物返潮、泥洗不掉、门口多水印、某句话被别人重复、旧绳结变紧、账本多出一行、界碑旁出现新脚印、门闩里侧多出手印。

## 第六阶段：后果渗出

后果像河泥里的水一样慢慢渗。递进顺序：物的异常 → 人的反应异常 → 家庭秩序异常 → 村庄规则被迫恢复 → 旁观者被卷入 → 真相证明后果已经开始 → 补救也成为传播的一环。

不要把后果写成单场追杀。

## 第七阶段：不可逆收束

结尾必须有代价。可用方向：旧物被送回去但送的人付出代价；规矩重新执行但某人不能再离开；真相被知道但村里更沉默；主角逃出村却带走某个痕迹；事件表面结束，传播链换了人。

禁止结尾：大师驱邪解决、主角开挂封印、一切都是梦、精神病解释一切、警方破案后一切恢复正常、神明显灵讲清全部因果。

## 单集短剧结构

适用于一集短剧：旧村日常、旧物出现、村民反应异常、老人阻止、嘴硬者破禁或差点破禁、小后果出现、旧事露出一个缺口、结尾钩子证明后果已经开始。

单集必须有一个旧物推进点、一个禁忌信息增量、一个村庄共谋瞬间、一个不可解释小后果、一个结尾钩子。

## 每章 / 每集检查

每章或每集至少推进一项：旧物状态变化、禁忌信息增量、旧事碎片、人物压力变化、村庄关系变化、后果扩散一步、主角卷入更深。

没有信息增量的恐怖戏，应删除或合并。
