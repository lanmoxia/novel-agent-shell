# 大纲示例

## 项目名

缺页名册

## 题材锁定

- 题材 Skill：huanghe-minsu-suspense-skill
- 类型：黄河故道 / 中式民俗恐怖 / 旧村禁忌 / 旧物污染
- 当前阶段：大纲示例
- 核心旧物：缺页名册

## 整体结构

村里重修旧祠堂，年轻村干部整理旧柜时发现一本缺页名册。名册记录的是当年清淤队的出工人名，但中间几页被撕掉，页边残留的泥印一直不干。老人要求把名册放回偏间，不许补页、不许抄名。村干部为了核对旧工程补助，把残名录进新表。第二天，新表里多出一个没人认识的名字，而那一户门口出现湿泥。随着清淤记录、河工后人和族谱缺口互相对不上，村干部发现当年清淤少的不是工钱，是人。最后名册被放回旧柜，缺页处却夹进了新打印的名单。

## 核心递进链

1. 日常旧村：重修祠堂，整理旧柜。
2. 异常旧物：缺页名册出现，页边泥印不干。
3. 村民避讳：老人不许补页、不许抄名。
4. 旧事露头：当年清淤队出工人数对不上。
5. 禁忌被破：村干部把残名录进新表。
6. 后果渗出：新表多名、门口湿泥、河工后人失态。
7. 不可逆收束：名册归柜，新名单却成了缺页的一部分。

## 章节 / 集数表

### 第 1 章：旧柜里的名册

- 核心事件：祠堂翻修，旧柜里发现缺页名册。
- 出场人物：村干部、守祠堂老人、木匠、河工后人。
- 主要场景：旧祠堂、偏间、院外旧堤。
- 旧物推进：名册出现。
- 旧物状态变化：缺页边缘泥印不干。
- 责任链变化：发现者被老人要求留下。
- 禁忌推进：不能补页，不能抄名。
- 恐怖信息增量：名册是当年清淤队出工册。
- 人物压力变化：村干部从办事变成被盯住。
- 章末钩子：名册缺页处夹着一小片新鲜湿泥。

### 第 2 章：不能抄的名字

- 核心事件：村干部核对补助，偷抄残名。
- 出场人物：村干部、老人、河工后人、会计。
- 主要场景：村委旧办公室、祠堂门口。
- 旧物推进：名册内容进入新表。
- 旧物状态变化：残名旁边浮出半个新字。
- 责任链变化：抄名者成为下一环责任人。
- 禁忌推进：观看类和传递类禁忌被破。
- 恐怖信息增量：当年出工人数和账本不一致。
- 人物压力变化：会计开始回避村干部。
- 章末钩子：新表里多出一个没人输入过的名字。

### 第 3 章：清淤记录

- 核心事件：旧清淤记录被找到，但日期和名册对不上。
- 出场人物：村干部、河工后人、老人。
- 主要场景：河工棚、旧堤、祠堂偏间。
- 旧物推进：名册与清淤记录互相冲突。
- 旧物状态变化：缺页处开始渗出旧墨痕。
- 责任链变化：河工后人承认父辈曾改过名。
- 禁忌推进：命名类禁忌浮出。
- 恐怖信息增量：少掉的人不是缺勤，而是被换名。
- 人物压力变化：村干部发现自己家姓氏也在残页边缘。
- 章末钩子：祠堂偏间门闩内侧出现泥手印。

### 第 4 章：放回去

- 核心事件：老人要求把名册放回偏间旧柜。
- 出场人物：村干部、老人、少数村民。
- 主要场景：旧祠堂、偏间、旧堤夜路。
- 旧物推进：名册被送回，但新表无法删除多出的名字。
- 旧物状态变化：缺页位置夹住一张新打印纸。
- 责任链变化：处理者发现补救也在扩散名单。
- 禁忌推进：送返类禁忌执行，但不等于结束。
- 恐怖信息增量：老人承认当年只是“把人从册上挪开”。
- 人物压力变化：村干部不再敢公开调查。
- 章末钩子：新打印纸上出现村干部的小名。

### 第 5 章：新缺页

- 核心事件：名册回柜后，村里恢复平静，但新名单变成旧册的一部分。
- 出场人物：村干部、老人、河工后人。
- 主要场景：祠堂偏间、村口、旧堤。
- 旧物推进：名册缺页被新纸补上。
- 旧物状态变化：新纸边缘也出现旧泥印。
- 责任链变化：村干部成为新的守册人。
- 禁忌推进：老规矩恢复。
- 恐怖信息增量：缺页不是缺失，是给后来的人留的位置。
- 人物压力变化：主角从调查者变成维护沉默的人。
- 不可逆代价：他知道真相，却不能再说全。
- 余味收束：祠堂门闩从里面轻轻响了一下。
