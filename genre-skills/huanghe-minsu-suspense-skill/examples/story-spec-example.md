# 故事企划示例

## 项目基本信息

- 项目名：旧渡口木牌
- 类型：黄河民俗恐怖 / 旧村禁忌
- 篇幅：短篇
- 当前阶段：故事企划示例

## 一句话梗概

返乡青年在旧堤清淤时挖出一块泥封木牌，牌上露出的半个旧地名让村里老人集体失声；他为了查清父亲当年离村原因读出完整地名，导致被抹掉的渡口重新出现在村民的日常路线里。

## 核心创意

- 黄河故道元素：旧堤、废渡口、被改名的老河槽。
- 旧村生活入口：雨后清淤，村民修排水沟。
- 民俗禁忌入口：旧地名不能喊全，木牌不能过门槛。
- 恐怖惊悚核心：木牌不是标识物，而是当年被村里故意撤掉的旧渡口记号。
- 观看者卷入方式：谁读出木牌上的完整地名，谁会在夜里听见渡口喊船。

## 核心旧物候选与选择理由

- 候选旧物 A：泥封木牌。
- 候选旧物 B：缺页族谱。
- 候选旧物 C：带泥钥匙。
- 最终选择：泥封木牌。
- 为什么不用常见旧物组合：本故事核心是旧地名和废渡口，不是器物内部异常；木牌比器皿、布料或水痕更能绑定“被撤掉的地点”。
- 该旧物与黄河故道 / 旧事 / 人物关系的绑定：木牌来自旧堤清淤，记录被废弃的渡口号，主角父亲当年参与撤牌。

## 主角设定

- 主角身份：返乡青年。
- 与村庄关系：少年时随父亲离开，成年后回来修旧宅。
- 初始动机：查父亲为什么不愿再提老家。
- 为什么会被卷入：他读出了木牌上被泥糊住的旧地名。

## 禁忌规则

- 明面规矩：旧渡口的名字不能喊全，木牌不能进门。
- 不能说全的部分：当年渡口不是荒废，而是被村里人共同撤掉。
- 谁维护这个规矩：几名老人和当年河工后人。
- 谁会破坏这个规矩：主角为了查父亲旧事读出地名。
- 破禁后的第一层后果：村口土路出现本不该通向旧渡口的湿脚印。

## 核心旧物

- 旧物名称：泥封木牌。
- 类型：木牌类。
- 来源：旧堤排水沟清淤时被铁锹碰出。
- 初始状态：木牌半截被泥封住，只露出一个旧地名偏旁。
- 谁发现：帮工河工。
- 谁碰过：帮工、主角、守规矩的老人。
- 状态变化：泥封裂开，旧渡口号逐渐露全。
- 不可替代性：只有木牌能把旧地名、渡口边界和村庄撤牌旧事绑定在一起。

## 恐怖递进

1. 日常旧村：返乡修宅，雨后清沟。
2. 异常旧物：泥下挖出木牌。
3. 村民避讳：老人不许读出完整地名。
4. 旧事露头：父亲当年参与撤掉渡口牌。
5. 禁忌被破：主角夜里擦掉泥封，读出旧地名。
6. 后果渗出：村路湿印、夜里喊船声、老人不再经过旧堤。
7. 不可逆收束：木牌被送回，但主角家门口多出一根渡船旧桩影子。
