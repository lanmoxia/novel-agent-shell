# 正文 / 剧本文风示例

雨停后，河工棚外的泥先泛亮。

老梁把胶鞋磕在门槛上，磕了两下，泥没掉干净。他骂了一句，弯腰去捡地上的出工牌。那牌子是从清淤沟里带回来的，木头发黑，边角被水泡得起了毛，背面还糊着一层细泥。

“旧牌子。”老梁说，“谁家孩子扔沟里的？”

没人接他的话。

棚里本来有三个人在喝茶，这会儿都看着他手里那块牌。最靠门的老周把茶缸放下，杯底碰到桌面，声音很轻。

“你从哪儿扒出来的？”老周问。

“东沟。”老梁把牌子翻过来，“铁锹碰着的。还挺硬。”

牌正面有两个字，被泥封住一半，只能看见一个“工”字边。下面还有一道浅浅的刻痕，像有人用刀尖反复刮过。

年轻的记工员伸手想拿。

老周先按住他的手。

“别过手。”

记工员愣住：“啥意思？”

老梁笑了一声：“一个出工牌，还不能过手？你们这帮老的，啥都能讲究。”

他说着要把牌子拍到桌上。老周忽然站起来，椅子腿在地上刮出一声响。

“我说别过手。”

棚里静了。

外头风从旧堤那边吹过来，雨布鼓了一下，又瘪下去。木牌上的泥沿着边角慢慢往下滑，露出背面一小块旧刻字。

记工员低头看了一眼，念出半个音：“陈……”

老周脸色一下变了。

“闭嘴。”

老梁的笑停在脸上。他低头看牌，手指不自觉往裤缝上蹭。那块出工牌在他掌心里湿得发冷，明明刚才已经擦过，背面的泥却又厚了一层。

记工员小声问：“这是谁的牌？”

老周没答。他先看了看门外，又看了看棚角那把清淤铁锹。铁锹刃口还靠着墙，黑泥滴在地上，一滴一滴，正好落成一条线，朝门槛外去。

“谁挖出来的？”老周问。

老梁说：“我。”

“那你今天别走。”

“凭啥？”

老周盯着他手里的牌，声音压得很低。

“当年少的不是一块牌。”

这句话一出来，棚里没人再动。记工员手里的本子翻到一半，纸页被风吹得哗啦响。他低头去按，发现自己刚记下的出工人数后面，不知什么时候多了一道湿泥印。
