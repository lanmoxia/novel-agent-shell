# 审查标准：黄河民俗恐怖题材检查表

## 使用方式

用于审查故事企划、大纲、正文、剧本片段是否符合黄河民俗恐怖题材。

审查结论分为：通过、有条件通过、不通过。

## 严重问题

出现任一严重问题，原则上不通过。

### 题材偏移

写成玄幻斗法、道士驱魔、神明显灵；写成普通刑侦推理，民俗只是外壳；写成血浆猎奇或怪物追杀；写成都市灵异爽文，失去黄河故道和旧村根基。

### 核心公式缺失

缺少以下多数要素：旧地貌记忆、村庄禁忌、被污染的旧物、老人不敢明说的旧事、诅咒式后果、现实质感下的不可解释感。

### 道具断裂

关键旧物凭空出现、凭空消失、状态前后不一致，或出现后不再推动剧情。

### 道具同质化严重

新项目在没有用户明确指定的情况下，默认复用固定旧物组合，导致不同故事只是换人名、换场景，旧物机制完全相同。

### 结尾糊弄

一切都是梦；精神病反转取消所有异常；大师突然驱魔解决；主角开挂封印一切。

## 中等问题

出现中等问题时，默认建议修订；是否作为非阻塞项接受并进入后续阶段，必须由用户明确确认，不得由 Agent 自动决定。

### 旧村生活质感不足

场景只写“阴森村庄”，缺少土墙、旧堤、河泥、饭桌、空院、清淤工具、人情关系等真实生活细节。

### 禁忌传播感不足

禁忌只停留在口头警告，没有通过观看、触碰、转述、处理旧物等方式扩散。

### 人物像功能 NPC

老人只负责讲设定，村民只会说“别问”，孩子像侦探，外来人为了作死而作死。

### 对白现代悬疑腔

人物频繁复盘、推理、讲证据链，缺少村里话、半句话、避讳和沉默。

### 恐怖靠跳吓

黑影、鬼脸、门响、尖叫过多，但没有信息增量和关系变化。

### 民俗解释过满

把禁忌写成完整神话体系或规则说明，失去不确定性。

### 道具同质化

新项目默认复用“红布 + 小铜碗 + 浑黄水 + 发丝”等固定高频组合，导致题材输出同质化。

## 轻微问题

个别句子有网文恐怖套话；场景氛围词略多，动作细节略少；对白个别句子偏书面；道具状态可再具体；旧物后果递进可再慢；某个配角的利益和恐惧不够清楚。

## 重点审查维度

### 题材符合度

是否明确属于“黄河故道 / 中式民俗恐怖 / 旧村禁忌 / 旧物污染 / 诅咒传播”。

### 旧村生活质感

是否有真实生活细节，而不是抽象恐怖布景。

### 禁忌传播感

禁忌是否通过人、物、话语、观看和责任链扩散。

### 悬疑推进

是否逐层揭露，而不是一次讲完或长期不提供信息。

### 道具连续性

旧物的来源、传递、状态变化和后果是否完整。

### 道具独特性

本项目核心旧物是否可以替换成任何别的旧物而不影响剧情？如果可以，旧物设计不合格。核心旧物必须和黄河故道、旧事、人物关系、禁忌规则形成不可替代绑定。

### 人物真实性

人物是否有身份、处境、利益、恐惧和差异化说话方式。

### 对白接地气

对白是否短、硬、土、有避讳，避免现代悬疑腔和大段设定解释。

### 恐怖方式

恐怖是否来自禁忌、旧事、旧物和后果，而不是廉价跳吓。

## 输出审查建议格式

1. 总结论：通过 / 有条件通过 / 不通过。
2. 严重问题：列出是否存在。
3. 中等问题：列出需要改的点。
4. 道具同质化判断：是否复用固定组合，旧物是否不可替代。
5. 轻微问题：列出可优化点。
6. 修改建议：按优先级给出 3-7 条具体建议。
7. 置信度：高 / 中 / 低。
