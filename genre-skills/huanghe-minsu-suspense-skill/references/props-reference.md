# 道具参考：旧物设计方法

## 旧物设计四问

每次设计核心旧物，先回答四个问题：

1. 从哪里来？它是从清淤泥里、旧堤下、祠堂偏间、废渡口、旧宅梁上、河工棚、旧柜、旧账本夹层，还是某户门槛下出现？
2. 谁拿过？谁第一眼看见，谁第一个碰，谁遮盖，谁转交，谁想丢掉，谁被迫送回？
3. 状态怎么变？泥封是否裂开，缺页是否浮字，绳结是否变紧，锁孔是否渗锈水，界碑是否位移，门闩是否出现手印？
4. 后果落到谁身上？挖出者、观看者、转述者、记录者、送返者、隐瞒旧事的人，谁成为下一环责任人？

答不出这四问，旧物不合格。

## 旧物类型池

### 木牌类

泥封木牌、河工出工牌、祠堂挂牌、旧渡口号牌。

### 纸册类

旧名册、缺页族谱、旧账本、清淤记录、未寄出的信。

### 器物类

铜铃、铜牌、铜扣、铜锁、铜镜残片、铜器残件、破搪瓷盆、旧木匣。

### 纺织类

旧绳结、褪色布条、麻绳、旧衣角、缝着名字的布片。

### 生活类

带泥钥匙、胶鞋、饭盒、旧水壶、煤油灯。

### 场景物类

废渡口木桩碎片、旧堤界碑、祠堂门闩、清淤铁锹。

### 污染物类

河泥、水痕、香灰、纸灰、黑水、头发、锈水。

污染物用于状态变化和证据，不应替代核心旧物。

## 旧物选择方法

按故事主题选择旧物：

- 旧地名和边界：选木牌、界碑、渡口号牌。
- 被抹去的人：选名册、族谱、旧照片、缝名布片。
- 河工旧债：选出工牌、旧账本、饭盒、胶鞋、铁锹。
- 封存空间：选铜锁、旧木匣、祠堂门闩、带泥钥匙。
- 观看禁忌：选铜镜残片、缺页记录、旧照片。
- 传递责任：选旧绳结、麻绳、褪色布条。

## 禁止固定组合规则

除非用户明确指定，不得默认使用固定高频组合。红布、铜碗、浑水、头发、旧包裹等都只是可选素材，不能绑定成默认公式。

如果每次故事都出现同一组旧物，Agent 会把题材误写成固定道具模板，而不是黄河民俗恐怖。

## 道具链示范方法

正确的道具链不是“拿出旧物吓人”，而是：旧物从具体地点出现 → 被具体人物接触 → 引发村民异常反应 → 状态发生小变化 → 牵出旧事碎片 → 责任链转移 → 后果落到活人身上。

## 旧物审查

检查：旧物能否被替换？如果能随便替换，说明它没有和旧事绑定。核心旧物必须承担不可替代功能。
